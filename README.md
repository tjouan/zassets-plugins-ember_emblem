zassets-plugins-ember_emblem
============================

[![Version][badge-version-img]][badge-version-uri]


  Ember with Emblem plugin for [zassets][].


Installation
------------

    $ gem install zassets-plugins-ember_emblem


Configuration
-------------

  zassets configuration:

``` yaml
plugins:
  - ember_emblem
```
